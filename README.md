# New Project Template

## Things to set up

- .env files in /client and /server
- Firebase service account key either in env or add firebase-sdk-key.json
- Package.json project name, version, package versions, repo link
- Package.json set-cors command
- CORS settings in server.ts
- Index.html title and favicon
- Node version
- Cursor rules for this project (Cmd shift P -> rules)

## Includes

- Clean file structure
- Ant Design and common building blocks
- Theme context
- Routing
- Firebase on client and server
- Vite config
- Lint and Prettier rules
- Gitignore
